# How to run this project
## 🪟 For Windows in cmd
```
git clone https://github.com/artin231/VPL.git
cd VPL
cd project
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
set FLASK_APP=app.py
flask run --debug
```

---

## 🐧 For Gnu/Linux in bash
```
git clone https://github.com/artin231/VPL.git
cd VPL
cd project
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
set FLASK_APP=app.py
flask run --debug
```
